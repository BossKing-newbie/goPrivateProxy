# Go语言私仓(goPrivateProxy)
<img src = "https://s1.ax1x.com/2022/06/27/jE2DYt.jpg" width = "50%" style="margin:0 auto">

一个基于[Goproxy](https://github.com/goproxy/goproxy) 的模块代理处理程序。
目标是通过构建私有仓库，实现maven的nexus的效果，节省企业带宽，对Go module进行管理和维护
## 1.特性
- 基于[Goproxy](https://github.com/goproxy/goproxy)
- 支持自定义Go module下载路径
- 支持文件树查看下载module
